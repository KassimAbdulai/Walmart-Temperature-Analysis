# Walmart Temperature Analysis
This project estimates the causal effect of temperature on weekly Walmart sales using store and time fixed effects.
The analysis is fully reproducible using the Quarto (.qmd) file provided.
## Files Included
- Wallmar_Data Exploration.qmd — Full data cleaning and regression analysis
- Walmart-Temperature-Analysis_project.pdf — Compiled project report
The analysis compares naïve and fixed effects models to estimate the short-run impact of temperature on sales.
